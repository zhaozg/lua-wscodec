Module ws.codec

Functions

decode (input[, offset=1])
Decode a WebSocket frame. Parses a WebSocket frame into a Lua table.

Parameters:

  • input string The WebSocket frame data to decode
  • offset number The offset in the input string to start decoding from (default is 1) (default 1)

Returns:

  1. parsed A Lua table containing the decoded frame data
  2. number The next frame offset in the input string

Or

  1. nil decoding fails
  2. string decoding error message
  3. number error code
encode (input, frame_type[, mask])
Encode a WebSocket frame. Generates a WebSocket frame based on the input data and frame type.

Parameters:

  • input string The data to encode
  • frame_type string The frame type, can be "text", "binary", "close", "ping", or "pong"
  • mask string|number The mask, can be a 4-byte string or a number (optional)

Returns:

    string The encoded WebSocket frame
